# 🚀 セットアップ手順（所要時間：15〜20分）

## ステップ1: GitHubアカウント作成（5分）

1. **https://github.com/signup** にアクセス
2. メールアドレスを入力 → 「Continue」
3. パスワードを設定（15文字以上 or 8文字以上で数字・小文字含む）
4. ユーザー名を決める（例: `taro-cams`）
   - **重要**: このユーザー名がアプリのURLに含まれます
   - 例: `https://taro-cams.github.io/cams-quiz/`
5. メール認証を完了
6. アンケートをスキップ（または適当に答える）
7. プランは「Free」を選択

✅ GitHubアカウント作成完了！

---

## ステップ2: リポジトリ（保管庫）を作成（3分）

1. GitHubにログインした状態で、画面右上の「**+**」→「**New repository**」をクリック

2. 以下のように入力:
   - **Repository name**: `cams-quiz`（好きな名前でOK、英数字・ハイフン推奨）
   - **Description**: `CAMS exam study app`（任意）
   - **Public** を選択（**重要：Privateだと無料Pagesが使えません**）
   - 「Add a README file」**チェック不要**
   - 「Add .gitignore」**None**
   - 「Choose a license」**None**

3. 「**Create repository**」をクリック

✅ リポジトリ作成完了！

---

## ステップ3: PWAファイルをアップロード（5分）

リポジトリのトップページが開いている状態から:

1. 「**uploading an existing file**」リンクをクリック
   - 「Quick setup」ボックス内にあります
   - またはURL末尾を `/upload/main` に変えてもOK

2. **重要**: ZIPファイル（`cams-pwa.zip`）を**展開してから**アップロード
   - Macなら：ZIPをダブルクリック → `cams-pwa` フォルダができる
   - その**フォルダの中身**（`index.html`、`app.js` など9個のファイル）を選択
   - フォルダごとではなく**ファイルだけ**をドラッグ&ドロップ

3. アップロード対象（9ファイル）:
   - ✅ `index.html`
   - ✅ `app.js`
   - ✅ `styles.css`
   - ✅ `manifest.json`
   - ✅ `service-worker.js`
   - ✅ `questions.json`
   - ✅ `icon-192.png`
   - ✅ `icon-512.png`
   - ✅ `apple-touch-icon.png`

4. 画面下部の「**Commit changes**」ボタンをクリック

✅ アップロード完了！

---

## ステップ4: GitHub Pagesを有効化（2分）

1. リポジトリの上部メニューの「**Settings**」（歯車アイコン）をクリック

2. 左サイドバーの「**Pages**」をクリック

3. 「Build and deployment」セクションで：
   - **Source**: `Deploy from a branch`
   - **Branch**: `main` を選択
   - フォルダ: `/ (root)` を選択
   - 「**Save**」をクリック

4. 数十秒〜2分待つ

5. 上部に緑色のバナーで以下のように表示されます：
   ```
   ✅ Your site is live at https://[ユーザー名].github.io/cams-quiz/
   ```

✅ デプロイ完了！

---

## ステップ5: iPhoneでアプリとして使う（3分）

### 📱 iPhoneのSafariで開く

1. iPhoneでSafariを起動
2. URLバーに以下を入力（自分のユーザー名に置き換え）:
   ```
   https://[ユーザー名].github.io/cams-quiz/
   ```
3. 数秒で「CAMS試験対策」のアプリが開きます

### 📲 ホーム画面に追加（重要！）

1. Safari下部の **共有ボタン** （□に↑のアイコン）をタップ
2. 下にスクロールして「**ホーム画面に追加**」をタップ
3. アプリ名「CAMS試験対策」を確認
4. 右上の「**追加**」をタップ

### 🎉 完成！

ホーム画面に **CAMS** アイコンが追加されました！

タップで起動すると：
- Safariのバーが消えてフルスクリーン表示
- アプリのように動作
- オフラインでも動く（一度開けば）
- 進捗は自動保存

# 🆘 トラブルシューティング

## ❓ Q: ページが表示されない（404エラー）

**A**: GitHub Pagesは反映に時間がかかります。
- 5〜10分待ってから再度アクセスしてみてください
- それでもダメなら、Settings → Pages で再度設定を確認

## ❓ Q: 問題が読み込まれない

**A**: ファイルが正しく配置されているか確認:
- リポジトリトップに `index.html` が直接あるか
- フォルダ内に入っていないか確認

## ❓ Q: iPhoneでホーム画面に追加が出てこない

**A**: 必ず**Safari**で開いてください。
- ChromeやFirefoxではこの機能は使えません
- 一度Safariで開いて「共有」→「ホーム画面に追加」

## ❓ Q: 進捗が消えてしまった

**A**: iOSのSafariは長期間使わないとデータを消すことがあります。
- 設定画面の「📤 学習データをエクスポート」で定期的にバックアップ
- 復元機能は次回アップデートで実装予定

## ❓ Q: ダークモードが切り替わらない

**A**: アプリ内の設定（⚙️）またはヘッダーの🌙ボタンから切り替え可能。

# 🌟 使い方のコツ

### 効率的な学習法

1. **毎朝5問**: 隙間時間に毎日続ける
2. **苦手分野を集中**: ホームの苦手TOP3から開始
3. **週末は模擬試験**: 120問の本番形式で実力チェック
4. **連続日数を伸ばす**: 1日1問でもOK、ストリーク維持

### 機能の活用

- 🔖 **ブックマーク**: あとで見返したい問題に付ける
- 🎲 **ランダム出題**: 「未解答のみ」で効率的に進める
- 💪 **弱点克服**: AIが自動で苦手な10問を選定
- 🔍 **検索**: キーワードで関連問題を一気に確認
